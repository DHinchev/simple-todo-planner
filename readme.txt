This is simple weekly todo planner written in React. The application is work in proggress and further features
will be added in the future to improve it.


How to run the application:

 1) Install Node.js (if you don't have it) and run npm install in the console
 
 2) Next do git clone - git clone https://github.com/DHinchev/simple-todo-planner.git

 3) Go in root folder and run npm start

 4) Wait for local server to load

 5) Enjoy
